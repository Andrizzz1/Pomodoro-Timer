# 🍅 Cute Pomodoro Timer

A simple and aesthetic Pomodoro web app designed to help you stay focused while keeping a cozy and cute vibe.

## ✨ Features

* ⏱️ 25-minute focus timer (Pomodoro)
* ⏸️ Play / Pause functionality
* 🔁 Reset timer
* 🎨 Clean and cute UI design
* 📱 Responsive layout (works on desktop & mobile)

## 🛠️ Built With

* React
* Tailwind CSS
* Heroicons
* Google Fonts (Nunito)

## 🎯 Purpose

This project was created to practice frontend development skills using React and Tailwind CSS.
It also aims to provide a simple productivity tool that is visually appealing and easy to use.

## 🚀 How It Works

1. Click **Play** to start the timer
2. Click **Pause** to stop it anytime
3. Reset when needed
4. Stay focused and repeat!
